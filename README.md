# About Kronos v1.2

**Kronos** is a high-performance, industry-leading UI library for Roblox script developers. Designed with a focus on **security**, **aesthetics**, and **developer experience**, Kronos provides a seamless bridge between your script's logic and a premium user interface.

## Why Kronos?

Kronos isn't just another UI library. It's a full-stack execution environment that handles the "boring" stuff so you can focus on your features.

- **Integrated Security**: Built-in Key System with Junkie SDK support, automatic HWID verification, and privacy-focused key masking.
- **Intelligent Loading**: A unique dual-phase yielding architecture that masks initialization lag.
- **Stealth Native**: Hardened internal architecture with randomized naming and environment masking.
- **Dynamic API**: Real-time property updates for all elements via the `:Update()` method.
- **Premium Logic**: Native integration for premium feature gating and manual locking.
- **Popout Panels**: Side-extending panels with console support, toggled from any button callback (v1.2).

## Core Philosophy

### Performance First

Kronos utilizes a light-weight instance management system. Your UI will maintain a rock-solid 60 FPS even on lower-end mobile devices, with layout updates calculated efficiently in the background.

### Developer Freedom

Write code linearly. No more complex callbacks or nested event handlers. `Kronos:CreateWindow` pauses your script until the user is ready, letting you write clean, readable Lua.
